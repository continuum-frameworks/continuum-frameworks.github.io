right-bottom

The right-bottom at-rule instructs the style-sheet to specifically style the right-bottom section of a printed page.

Definition​

CSSContinuum
@right-bottom { style-sheet }@r042[style-sheet]

General Usage​

HTML
<!-- Page Rule -->
<div class="@at-rule"></div>

Therefore, when expecting:

CSS
/* Example 1 */
@page {
    @right-bottom {
        content: "Page " counter(pageNumber);
    }
}

Respectively use:

HTML
<!-- Example 1 -->
<div class="@p045[@r042[c204=['Page_',c434(pageNumber)]]]"></div>

For more information on how to implement CSS at-rules as defined by Continuum, refer to the Properties documentation.
