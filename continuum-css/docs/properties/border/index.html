border

The border property controls an element's border.

Overview​

Given the following CSS styling for example:

Example CSS styling
/* Global Keyword */
border: inherit;
border: initial;
border: revert;
border: revert-layer;
border: unset;

The corresponding Continuum classes to use would be:

Example HTML use
<!-- Global Keyword -->
<div class="b423=i829"></div>
<div class="b423=i631"></div>
<div class="b423=r486"></div>
<div class="b423=r319"></div>
<div class="b423=u558"></div>

View the Prime Example section at the end of this page for a more comprehensive look into using Continuum.

Basic Usage​

Values​

To set the border property using Continuum value classes:

Example HTML use
<!-- Keyword Color -->
<div class="b423=c282"></div>

<!-- Named Color -->
<div class="b423=r054"></div>

<!-- Hexadecimal Color -->
<div class="b423=#0f0"></div>

<!-- System Color -->
<div class="b423=a568"></div>

<!-- Length Unit -->
<div class="b423=110vh"></div>

<!-- Keyword Color, Named Color, Hexadecimal Color, System Color, Length Unit (mixture) -->
<div class="b423=c282,d564,110vh"></div>

For a full list of all currently supported global, keyword, and other named CSS values, view the Values documentation.

Value-functions​

To set the border property using Continuum value-function classes:

Example HTML use
<!-- Color Function -->
<div class="b423=h459(240,100%,50%/1.0)"></div>

For a full list of all currently supported global, keyword, and other named CSS value-functions, view the Value-functions documentation.

Advanced Usage​

Conditional via At-rules​

To conditionally set the border property using Continuum at-rule classes:

Example HTML use
<!-- Media Rule -->
<div class="@m074(p999)[body[b423=s705]]"></div>

For a full list of all currently supported global, keyword, and other named CSS at-rules, view the At-rules documentation.

Conditional via Pseudo-classes​

To conditionally set the border property using Continuum pseudo-class classes:

Example HTML use
<!-- Interaction Class -->
<div class="b423:f194=d564,r054"></div>

For a full list of all currently supported global, keyword, and other named CSS pseudo-classes, view the Pseudo-classes documentation.

Conditional via Pseudo-elements​

To conditionally set the border property using Continuum pseudo-element classes:

Example HTML use
<!-- Cosmetic Element -->
<div class="b423::a205=t998,d182,#32a1ce"></div>

For a full list of all currently supported global, keyword, and other named CSS pseudo-elements, view the Pseudo-elements documentation.
