Installation

Continuum works by scanning for class names in all of your project's HTML files and JavaScript components, then generating and adding the corresponding styles to a dynamically created <style> tag.

This short process is fast and reliable, with the only runtime variable being the client itself.

Requirements​

None whatsoever.

Currently, there's only one way to integrate Continuum into your project:

  • By including it via a <script> tag referencing the version you want.

This process doesn't necessitate any requirements on your part as the developer.

Steps​

Step 1​

Add the following <script> tag to the <head> element of your HTML file:

HTML
<!DOCTYPE html>
<html>
    <head>
        ...

        <script
            defer
            src="https://cdn.jsdelivr.net/npm/continuum-css@latest/dist/cdn.min.js"
        ></script>
    </head>

    ...
</html>

Notice the use of @latest in the provided <script> tag. This ensures that only the latest version of Continuum is pulled from the CDN.

tip

Continuum is designed to always be stable for production.

However, if the latest version has breaking changes you don't want to deal with via upgrade guides, or whether you just like the version you're currently using, you may choose to hardcode the version you want in the CDN link.

For example, to use the latest version 1 of Continuum:

<script
    defer
    src="https://cdn.jsdelivr.net/npm/continuum-css@1.x.x/dist/cdn.min.js"
></script>

Or, to use a specific version 1 of Continuum:

<script
    defer
    src="https://cdn.jsdelivr.net/npm/continuum-css@1.0.0/dist/cdn.min.js"
></script>
warning

Don't forget to add the defer attribute to your <script> tag.

Without it, Continuum will not work as expected, or may even not work at all.

note

The rest of the steps below are optional as Continuum is now ready to use.

Step 2 (Optional)​

Add some Continuum classes to your elements in order to style your content:

HTML
<!DOCTYPE html>
<html>
    ...

    <body>
        <h1 class="f582=30px f673=750 t138=u441">Hello World!</h1>

        ...
    </body>
</html>

That's it! Continuum will now style your tag's content as required at runtime.

tip

Continuum is designed to have a low barrier of adoption.

To that end, you can pick the level of complexity in syntax that you want to use in your projects.

For example, to use the simplest possible syntax:

<h1 class="font-size=30px ...">Hello World!</h1>

Or, to use a syntax some notches above that:

<h1 class="... font-w966=750 ...">Hello World!</h1>

Or even to mix and match the syntax as you wish:

<h1 class="... text-d549-line=u441">Hello World!</h1>

All of these options are perfectly valid as it only depends on your desire as the developer.

What's Next?​

You may choose to get familiar with some of the Concepts that underpin Continuum, or to dive straight into exploring all of the applicable syntax through the Syntax documentation.

Whatever you decide, Continuum is installed and ready to go.
